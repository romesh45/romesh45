### 🔷 ZenHire — AI Hiring Platform *(Active Build)*
> Production SaaS · 7 AI Agents · Multi-Tenant · Stripe Monetised

End-to-end AI hiring pipeline. Resume intelligence → adaptive interview → hallucination detection → deterministic decision. Decision Engine uses **zero LLM** — pure deterministic Python. Every decision is auditable and legally defensible.

**What makes it different:** Hallucination detection catches experience bluffs. Penalty carries forward to reduce the resume score. Dual-layer explainability — full audit trace + human-readable recruiter summary, derived deterministically.

```
Python 3.12 · FastAPI · LangGraph · GPT-4o · LiteLLM · Qdrant
PostgreSQL 16 · Redis 7.2 · Celery · Next.js 14 · TypeScript
Docker Compose → Kubernetes + Helm + ArgoCD · Stripe · Resend
```
